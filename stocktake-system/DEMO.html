5. DAT File Export (For HnL Import)

This is what gets exported for importing back into HnL:

9320000500579   36
9313419522225   18

Format Rules:

  • Barcode starts at position 1
  • Count starts at position 17 (padded with spaces)
  • Only includes items with barcodes
  • Only includes items with non-zero counts
  • Excludes items without barcodes (like kegs)

7. Audit Trail

Automatic tracking of all manual adjustments:

Key System Behaviors

  • Negative Theoretical: Handled normally (e.g., -270.87 becomes +270.87 variance if counted as 0)
  • Multiple Scans: Automatically summed by barcode (24 + 12 = 36)
  • No Barcode: Shown in blue, excluded from .dat export
  • Manual Override: Latest manual entry replaces scanned counts
  • Real-time: Updates immediately when you click save
